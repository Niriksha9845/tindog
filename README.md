# 🐶 TinDog

A responsive landing page for a fictional dog dating app built using HTML, CSS, and Bootstrap 5.

> **Note:** This project is a clone of the TinDog project from **Dr. Angela Yu's Complete Web Development Bootcamp**. It was built as part of my learning journey to practice Bootstrap and responsive web design.

## 🚀 Features

- Responsive layout using Bootstrap 5
- Bootstrap Grid System
- Navigation Bar
- Hero Section
- Features Section
- Testimonials
- Pricing Cards
- Footer

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 5
